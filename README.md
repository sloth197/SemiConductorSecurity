# SemiConductorSecurity
 반도체 장비 보안과 이상 동작을 통합해 탐지하는 시스템 프로그램

## 개요
- 반도체 장비에서 발생할 수 있는 보안 위협 + 센서 기반의 이상 동작을 감시할
  수 있는 모니터링 시스템입니다.
- 반도체 장비는 공정, 센서, 파일 등 다양한 요소가 결합되어 동작하기 때문에
  보안 취약점과 공정의 이상이 함께 발생할 수 있습니다. 이 프로젝트는 그러한 두 단점을 함께 모니터링하며 실시간으로 표시합니다.

## 핵심기능

### 1. 실시간 센서 데이터 표시
- 온도, 압력, 속도, 가스 유량
- 장비 내부의 센서 값 시뮬레이션
- 1초 주기 실시간 업데이트

### 2. 센서 이상 감지
 - 온도 > 220
 - 속도 > 1800
 - 가스 유량 < 15
 - 압력 < 0.9
 -> 비정상적인 값을 감지할 때 치명적(Critical) 경고 표시중

 ### 3. 보안 모듈
 - 파일 무결성 검사 -> 파일 변경 감지 (SHA256)
 - 악성 및 의심 프로세스 감지
 - 비정상 명령 실행 탐지

 ### 4. 보안 + 공정 통합 판단 시스템
 -  파일 변경이 감지되고 온도가 200도 이상일 때 치명적(Critical) 경고 알림
